Orçamento de Viagem

Este é um aplicativo desenvolvido em Python que criei como exercício para aprendizado e também estímulo criativo

Resumo

Este projeto é muito interessante por sua simplicidade e praticidade. Com ele é possível colocar todas as despesas da viagem que se pretende fazer, colocar o valor que se pretende gastar e manipular esses dados a vontade, reparando erros ou fazendo simulações. Após tudo isso existem as opções de baixar a imagem do gráfico em PNG ou salvar a lista de despesas em um arquivo Excel.

Principais bibliotecas utilizadas

  • TKINTER
  • MATPLOTLIB
  • PANDAS
  • SQLITE3
